# SecLeaf Writeups

A curated collection of Capture The Flag (CTF), Digital Forensics & Incident Response (DFIR), OSINT, Web Security, Binary Exploitation, and Reverse Engineering writeups solved during cybersecurity competitions and training exercises.

## About

This repository documents my methodology, investigation process, tooling, and lessons learned while solving security challenges.

Each writeup contains:

* Challenge description
* Reconnaissance methodology
* Investigation process
* Technical analysis
* Screenshots and evidence
* Solution walkthrough
* Lessons learned
